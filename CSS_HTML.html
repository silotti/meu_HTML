Sou um título CSS inline

Ao utilizar o seletor de elementos, todos os elementos serão afetados pelo estilo CSS, ou seja, todos os parágrafos terão as mesmas propriedades CSS.

Sou um paragrafo que usa o estilo CSS interno de "p", mais como também tenho uma id, sou afetado pelo estilo via seletor de elementos, então meu estilo usado é do "id".

Subtitulo com classe utilizada também em outro elemento

Oi, não se esqueça de mim. Mesmo estando dentro de outro elemento, sou afetado pelo estilo aplicado via seletor.
